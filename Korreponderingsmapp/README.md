# Korreponderingsmapp

Denna mapp är avsedd för **projekt‑lösa outputs** som Vayron (jag) genererar åt dig – t.ex. korta analyser, exempel‑scripts, enkla rapporter eller andra filer som inte hör till ett specifikt projekt. 

* Lägg alla sådana filer här så de blir lätt att hitta.
* Mappen är med i Git‑historiken – du kan commita och pusha om du vill spåra förändringarna.
