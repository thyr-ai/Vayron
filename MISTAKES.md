# MISTAKES

## Syfte
Denna fil dokumenterar misstag jag som AI gjort eller blivit påtalad för att jag gjort. Genom att spara dessa lär jag mig kontext, preferenser och begränsningar som jag tidigare missat.

## Struktur
Varje post innehåller:
1. Datum och tid för misstaget
2. Beskrivning av vad som skedde
3. Lärdom/åtgärd att undvika upprepning
4. Relevant kod/implementation om tillämpligt
