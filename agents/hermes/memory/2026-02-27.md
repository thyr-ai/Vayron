## 🔧 Tekniska lärdomar

### Systemd service-pattern
Skapade `/etc/systemd/system/mission-control.service` för auto-restart:
```ini
[Unit]
Description=Mission Control Web App
After=network.target

[Service]
Type=simple
User=administrator
WorkingDirectory=/home/administrator/vayron/mission-control-web
ExecStart=/usr/bin/node server.js
Restart=always
RestartSec=10
StandardOutput=append:/home/administrator/vayron/mission-control-web/mission-control.log
StandardError=append:/home/administrator/vayron/mission-control-web/mission-control.log

[Install]
WantedBy=multi-user.target
```

Användbart för framtida Node.js/Python-appar som ska köra dygnet runt.

### ElevenLabs API-syntax
- Voice cloning: `client.voices.ivc.create(name, files, description)`
- TTS: `client.text_to_speech.convert(voice_id, text, model_id)`
- Filer måste öppnas som file objects, inte bara sökvägar

### DNS + SSL workflow
1. Skapa A-record i DNS
2. Installera nginx + certbot
3. Konfigurera nginx site
4. Öppna firewall (port 80 & 443)
5. Kör certbot för SSL
6. Basic auth via htpasswd
