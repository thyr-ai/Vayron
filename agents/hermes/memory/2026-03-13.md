## Chrome Extension Relay - Felsökning (misslyckad)

### Bakgrund
- **Syfte:** Ge Vayron access till Mattias Chrome-tabbar (särskilt Byggnads A-kassa)
- **Teknologi:** OpenClaw Browser Relay extension via CDP port 18792

### Troubleshooting Timeline (13:12 - 14:57 UTC)
1. **Första försöket:** Comet browser
   - Extension visade "disconnected" och "relay not running"
   - Upptäckte att Comet inte är Chrome/Chromium
   - Mattias vägrade byta till Chrome → pausad

2. **Andra försöket:** Äkta Chrome (14:48 UTC)
   - SSH tunnel: `ssh -L 18792:127.0.0.1:18792 administrator@85.190.102.252`
   - Extension config: Port ändrad från 18789 → 18792
   - Relay reachable (grön text i options)
   - Extension visade "attached" (orange ON badge)
   - **Problem:** Gateway såg 0 tabbar (`browser tabs chrome` returnerade tom array)

3. **Felsökningssteg genomförda:**
   - Verifierade SSH tunnel (netstat visade anslutningar på 18792)
   - Gateway restart (pkill + nohup) → SSH tunnel dog, måste startas om
   - Attach/detach cycle flera gånger
   - Aldrig löst - gav upp efter 30+ minuter

### Lessons Learned
- **Chrome Extension Relay är krånglig** över SSH tunnel
- **Relay sync issue:** Extension säger "attached" men Gateway får inga tab-events
- **Bättre alternativ:** Screenshots eller copy-paste för inloggade sidor
- **Framtida försök:** Överväg Tailscale Serve eller lokal Gateway på Mac
