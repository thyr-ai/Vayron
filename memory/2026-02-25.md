## Morgon - Telegram-gruppkonfiguration

**Problem:** Telegram-gruppkanaler kom inte igenom till mig
- Mattias ville öppna sina 4 Telegram-kanaler (Professional, Semiprofessional, Personal, Private)
- Jag försökte först med `allowGroups` (som inte existerar)

**Lösning:**
- Läste OpenClaw docs: `/channels/telegram.md`
- Rätt syntax är att definiera varje grupp under `channels.telegram.groups` med grupp-ID som nyckel
- Konfiguration som fungerade:
```json
"channels.telegram.groups": {
  "-5052479766": {"groupPolicy": "open", "requireMention": false},  // Professional
  "-5065314519": {"groupPolicy": "open", "requireMention": false},  // Semiprofessional
  "-5020563698": {"groupPolicy": "open", "requireMention": false},  // Personal
  "-5282965539": {"groupPolicy": "open", "requireMention": false}   // Private
}
```

**Lärdom:** 
- ❌ Gissa aldrig OpenClaw-syntax
- ✅ Läs alltid docs först: `~/.npm-global/lib/node_modules/openclaw/docs/`
- Lade till permanent guidelines i AGENTS.md
