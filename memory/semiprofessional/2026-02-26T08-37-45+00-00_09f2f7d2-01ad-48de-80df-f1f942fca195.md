# Projekt: Andra OpenClaw-instans på VPS

**Syfte:** Sätta upp en andra OpenClaw-instans på samma VPS som Mattias kan hjälpa någon med låg teknisk kunskap att använda.

**Setup-plan:**

1. **Separata instanser:**
   - Vayron: `/home/administrator/vayron` (port 3000)
   - Ny användare: `/home/administrator/[namn]` (port 3001)

2. **Isolation:**
   - Egna workspaces
   - Egna configs
   - Egna agents

3. **Mattias hjälper via Vayron:**
   ```bash
   # Mattias ber Vayron:
   "Gå in i [namn]s OpenClaw och fixa X"
   
   # Vayron kör:
   cd /home/administrator/[namn]
   openclaw chat "gör uppgift Y"
   ```

4. **Fördelar:**
   - Full isolation mellan användare
   - Mattias kan hjälpa utan att användaren behöver teknisk kunskap
   - Vayron kan agera admin/support
   - Användarens data stannar i deras workspace

**Status:** Planering/koncept

**Nästa steg:** Väntar på att Mattias bestämmer vem som ska få instansen.
