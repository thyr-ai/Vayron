## Fulltext
Mattias vill visa sina träningsrekord från Strava på ovning.se-hemsidan. Jag har skapat komplett dokumentation i STRAVA_API.md med:

**API-kunskaper:**
- OAuth 2.0 autentisering (access token + refresh token)
- Rate limits: 200/15min, 2000/dag
- Endpoints: /athlete/activities, /activities/{id}, /athletes/{id}/stats

**Implementation-plan:**
1. Skapa Strava App på strava.com/settings/api
2. OAuth-flow för att få refresh token
3. Python script för att hämta aktiviteter
4. Analysera data för rekord (längsta run, snabbaste 10k, etc)
5. Exportera till JSON
6. Visa på ovning.se med HTML/JS

**Automation:**
- Cron job dagligen kl 06:00 för att uppdatera data
- Auto-refresh av access tokens (går ut var 6:e timme)
- Rate limit hantering med retry-logik

**Credentials:**
- Ska sparas i credentials/strava-api.json (git-crypt krypterad)
- Client ID, Client Secret, Refresh Token

**Nästa steg:**
- Väntar på att Mattias skapar Strava App
- Hjälpa med OAuth-flow
- Bygga fetch-script när credentials finns
